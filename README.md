# telegram-stats-bot

## Deploy

Есть два стула

### Без Docker

Настроить переменные окружения, и подготовить пользователя MySQL заранее

```bash
npm i
npm i -g sequelize
sequelize db:create
sequelize db:migrate
npm run start
```

### Docker

```bash
docker-compose up --build --detach
```
